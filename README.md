# Első labor

## A labor teljesítése
Ebbe a könyvtárba készítsd el az **android labor** megoldásait. Az extra feladatok megoldásával pontok gyűjthetők a megajánlott jegyhez. A feladatok részletes leírása a [https://bmeviaubxav082.github.io/tudnivalok](https://bmeviaubxav082.github.io/tudnivalok/laborok/) oldalon szerepel.

> [!IMPORTANT]
> A megoldásokat pull request formájában kell beadni a határidő előtt a feladatkiírásban szereplő [Feladatok beadása](https://bmeviaubxav082.github.io/tudnivalok/github/GitHub/) leírásban található utasítások alapján.
> - Hozz létre egy **új branchet** `megoldas` néven, és ezen dolgozz.
> - Töltsd ki a `neptun.txt` fájlt a saját Neptun kódoddal.
> - Minden feladat után kommitolj, és használj értelmes kommit üzeneteket.
> - A feladat végeztével vagy a labor végén **pushold** a megoldásodat és hozz létre egy **pull requestet**.
> - Ellenőrizd a pull request tartalmát és rendeld hozzá a laborvezetődet **reviewernek**.

> [!CAUTION]
> A nem ilyen formában beadott megoldások nem lesznek értékelve!
